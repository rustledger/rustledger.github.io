Accessibility Statement

We are committed to making rustledger accessible to everyone.

Accessibility Features

  • Keyboard Navigation: All interactive elements are keyboard accessible. Use Tab to navigate and Enter to activate.
  • Focus Indicators: Visible focus outlines on all interactive elements.
  • Skip Link: "Skip to playground" link for keyboard users to bypass navigation.
  • ARIA Labels: Screen reader support for buttons, tabs, and dynamic content.
  • Color Contrast: Text meets WCAG AA contrast requirements.
  • Semantic HTML: Proper heading hierarchy and landmark regions.

Keyboard Shortcuts

  • ? - Show keyboard shortcuts help
  • Ctrl + Shift + F - Format code
  • Enter - Run query (when in query input)
  • Esc - Close modals

Known Limitations

  • The code editor (CodeMirror) has its own accessibility features but may require screen reader configuration.
  • Query result tables with many columns may require horizontal scrolling on mobile.

Feedback

If you encounter accessibility barriers or have suggestions for improvement:

  • Open an issue at GitHub Issues
  • Use the contact form on the main page

Standards

We aim to conform to WCAG 2.1 Level AA guidelines.
